# ultrasonic.distance_cm(PORT) < x
![ultra 2](/images/app/block-api/api-input/images/ultra-2.png)


Trả về giá trị `True` hoặc `False` khi so sánh khoảng cách đo được từ cảm biến khoảng cách tới vật thể đổi diện với một giới hạn cho trước (đơn vị `centimet`)


## Ví dụ
Thay đổi màu đèn trên bo mạch nếu cảm biến khoảng cách phát hiện vật cản dưới 10 cm
