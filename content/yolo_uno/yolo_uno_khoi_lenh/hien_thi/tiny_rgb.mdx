# 1. Đổi màu đèn Tiny RGB

## 1. Mục tiêu
Viết chương trình điều khiển module LED RGB thay đổi thành các màu khác nhau

## 2. Thiết bị cần sử dụng
- Mạch Yolo UNO:
![yolo uno](/images/yolo_uno/yolo_uno_khoi_lenh/hien_thi/images/yolo_uno.png)


- Module led RGB kèm dây tín hiệu:
![tiny rgb](/images/yolo_uno/yolo_uno_khoi_lenh/hien_thi/images/tiny_rgb.png)


## 3. Kết nối phần cứng
Kết nối module led RGB vào chân D3 của Yolo UNO
![tiny rgb 1](/images/yolo_uno/yolo_uno_khoi_lenh/hien_thi/images/tiny_rgb_1.png)


## 4. Chương trình lập trình
- **Giới thiệu các câu lệnh:**
![tiny rgb 2](/images/yolo_uno/yolo_uno_khoi_lenh/hien_thi/images/tiny_rgb_2.png)


Bạn có thể lựa chọn chân điều khiển, màu và các led trên module sáng khác nhau.

- **Chương trình lập trình:**
![tiny rgb 3](/images/yolo_uno/yolo_uno_khoi_lenh/hien_thi/images/tiny_rgb_3.png)


## 5. Chương trình mẫu
Nhấp vào chữ tại đây để xem chương trình mẫu, hoặc quét mã QR bên dưới để xem chương trình.
