# Robot ORC

Nhóm này tổng hợp các lỗi thường gặp khi sử dụng robot ORC, bao gồm:

- không mở được code mẫu hoặc thiếu khối lệnh
- lỗi encoder và lỗi cảm biến góc
- robot chạy lệch, chạy sai hướng
- lỗi ORC Hub và lỗi ghép nối tay cầm
- lỗi khi dùng module 8 kênh servo
